# Trackline

Trackline is an app for tracking internship applications — from the first time you apply
all the way through interviews and offers.

It keeps everything about your internship search in one place: which companies you've applied
to, what stage each one is at, interview rounds and notes, contacts, resumes and cover letters
you used, and a dashboard that shows your overall progress (response rate, interview rate,
offer rate, and more).

## Live app

- **App:** https://intership-applications-tracking.vercel.app
- **API:** https://trackline-api.onrender.com

The API runs on a free hosting tier, so it may take 30–60 seconds to respond on the first
request after a period of inactivity.
